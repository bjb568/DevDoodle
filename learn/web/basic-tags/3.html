Introduction

In this section, we're going to learn about the basic tags that are the building blocks of HTML.

P is for Paragraph

One of the most basic and fundamental tags in HTML is the p tag. P is (really) short for paragraph. As shown in lesson 3, it looks like this:
<p>This is a paragraph.</p>

To move on, write a set of paragraph tags in between the body tags. You can put any text you want in them.
