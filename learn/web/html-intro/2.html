Setting up a page - Using a Doctype

The first thing you do when you write an HTML page is define the doctype.

<!DOCTYPE html>

This tells the browser that the page is in HTML5. There are other doctypes, but right now you only need to know the doctype of HTML5. Type it in the code area to continue.
