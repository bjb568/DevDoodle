Intro to HTML

Recap

Congratulations! You now know what HTML is. Let's review:
  • HTML stands for Hyper Text Markup Language - it's a language used to mark up text on the web.
  • HTML is used to provide formatting, layout, and a foundation for other web technologies.
  • You mark up text with tags - a start tag before the content and an end tag after.
  • The basic HTML document consists of a doctype and html, head, and body tags: <!DOCTYPE html> <html> <head></head> <body></body> </html>
Next, we'll learn more about tags, and how to format text.
