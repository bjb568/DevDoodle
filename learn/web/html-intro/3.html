Setting up a page - Tags

HTML applies semantics and formatting using tags. In the first lesson, we saw a few tags, including the html, head, body, and p tags.

What is a tag?

There are two types of tags - start tags (<p>) and end tags (</p>) enclosed in brackets (all start-tags have end-tags). To write a start tag, type < (less-than), the tag name, then > (greater-than). To write an end tag, type < (less-than), a / (slash) the tag name, then > (greater-than).

<p></p>

The tag contents (anything between the start and end tags) are formatted differently depending on what the tag is.

Right now, we need to set up a webpage, so we need to use some tags.

<!DOCTYPE html> <html> <head></head> <body></body> </html>

After the doctype, there is an <html> tag. Inside it is <head> and <body>. The head tag will contain all meta-tags (title, description, etc.), while the body tag will contain visible content.
