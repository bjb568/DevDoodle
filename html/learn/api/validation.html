In the textarea given, you must define a function validate(String data). It is passed a string parameter data containing the contents of the user's code <textarea>.

Return object

This function is to return an object containing properties:

  • success: Boolean of whether the code is a correct solution (default true)
  • msg: String shown to the user that describes the mistake or a congradulation (default "You did it!")

Example

This is the default code given.

function validate(data) { if (data.indexOf('test') == -1) { return { success: false, msg: 'Did you remember to foo the bar?' }; } return { success: true, msg: 'You did it!' }; }

This will pass any code containing the string "test", otherwise it will show the message Did you remember to foo the bar?. Note that the second return statement is optional.
