Basic Tags

"H" is for Heading

Heading elements outline the structure of a document. HTML provides 6 levels of headings: <h1> for the page title, <h2> for the section title, <h3> for the subsection title, and so on down to <h6> for the subsubsubsubsection title. ideally, these headings could be used to create a table of contents.

To move on, write all the possible header tags from most to least important and give each of them some text content.
