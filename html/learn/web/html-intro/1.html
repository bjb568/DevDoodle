What is HTML?

HTML (Hyper Text Markup Language) is the language of web pages. It augments text by adding semantic meaning, organizing bits of text into headers, paragraphs, hyperlinks (text that links to another document), and other elements, and serves as a foundation for other web technologies such as CSS, which provides style, and JavaScript, which enables interactivity.
