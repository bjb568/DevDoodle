Intro to HTML

First off, Use a Doctype

The first thing that you must do to write an HTML document is to define the doctype.

<!DOCTYPE html>

This tells the browser that the page is in HTML5 (the latest version of HTML). There are other doctypes, but we will just be using this one for this tutorial.
