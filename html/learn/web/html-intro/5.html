Intro to HTML

Recap

Congratulations! You now know the basics of HTML. Let's review:

  • HTML stands for Hyper Text Markup Language — it's used to give semantic meaning to plain text on the web.
  • HTML centers around elements which you write using tags.
  • Text is wrapped between a start tag and an end tag, where only the end tag has a slash
  • The basic HTML document consists of a doctype, a root html element, and head and body sections. Metadata like the page title goes in the head element, while visible content like paragraphs go in the body element

Next, we'll learn more about tags, and how to format text.
