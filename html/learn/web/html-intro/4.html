Intro to HTML

Lets add some content!

So far we have:

  • a doctype declaration to tell the browser we are using HTML5
  • an html element that contains all other elements
  • a head element that will contain document metadata
  • and a body element that will contain the page content

Lets bring our page to life by adding a title and some text.

The title element

Every HTML document should have a title that describes what the document is about. Titles are put inside the head tag:

… <head> <title>My awesome webpage</title> </head> …

Add one to your document!

How about some content?

As was mentioned before, the paragraph element contains regular text and (<p></p>) can be put inside the body element. Add a paragraph to your document!
