Links

Links are the binding force of the web. They allow you to navigate from one page to another just by pressing one. To create a link in HTML, we use the <a> tag. The a tag has an attribute called href that specifies where we want the link to go to. If we want the link to go to http://placekitten.com we use the attribute href="http://placekitten.com". And so, our full tag would look like this:

<a href="http://placekitten.com">When someone clicks on this text, they will be sent to placekitten.com!</a>

To move on, write a link to https://devdoodle.net with any text you want.
