Links

Links are the binding force of the web. They allow you to navigate from one page to another with just a tap. To create a link in HTML, we use the <a> (anchor) tag. The a tag has an attribute, href (Hypertext Reference), that specifies where we want the link to go to. If we want the link to go to http://example.com (the "http" part is important) we use the attribute href="http://example.com". Our full tag would look like this:

<a href="http://example.com">click here!</a>

This appears as: click here!

An attribute defines a property of an element on its start tag (attributes always go on start tags, not end tags). They specify additional information that may change the way the element behaves, in this case where the link goes to. The text inside the attribute isn't seen by the user, but is seen by the browser.
