Emphasis

Sometimes when we're writing, we want to give a bit more emphasis to our words. With HTML there are two tags for that:

  • <em></em> for emphasis, usually rendered as italic
  • <strong></strong> for strong emphasis, usually rendered as bold

To move on, use first the <em> element, then the <strong> element. Both can contain any text you want.
