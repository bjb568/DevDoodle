Basic Elements

Recap

You've learned the basic elements needed to create a webpage. Let's review them:

  • <p></p> is for paragraphs, where most of your text will go
  • <h1></h1> to <h6></h6> are headings from most to least important
  • <a></a> creates a hyperlink that connects webpages together
  • <img /> is a self-closing tag, and it displays and image from an external source
  • <em></em> and <strong></strong> mark text with emphasis, and you can put them inside a paragraph
  • <!-- --> is a comment which users don't see, but can help you stay organized while you write HTML

Next, we'll learn how tags can work together.
