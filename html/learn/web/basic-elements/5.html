Images

Don't we all love looking at pictures of adorable kittens and puppies on the internet? Well, the <img /> tag lets us have these! Similar to the <a> tag, images take an attribute, src (source), which points to the url of the image. Unlike most tags, the <img> tag is self-closing, since it doesn't have text content. There is only one image tag that acts as both the start and end tags. The tag looks like this:

<img src="url" />

Notice that it has a src attribute and a slash at the end.
