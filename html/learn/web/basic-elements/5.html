Images

Don't we all love looking at pictures of adorable kittens and puppies on the internet? Well, this wouldn't be possible without the <img> tag! The <img> tag allows us to display images on our webpage. Just like the <a> tag, it takes an attribute src (which is just an abbreviation for source) which points to the url of the image. Unlike most tags, the <img> tag is self closing, so it only includes one tag with a /> on the end of it. The syntax for the <img> tag looks something like this:

<img src="url"/>
