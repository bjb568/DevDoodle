Comments

What if you wanted to write something in the HTML just for yourself, something that wouldn't get shown by a browser? That's what comments are for. Altho comments don't affect what the end user will see, it helps you, the developer, stay organized. Unlike most HTML tags, comments start with <!-- and end with -->. You can put anything in a comment (except an end comment tag).

<!-- This is what a comment looks like. It doesn't functionally do anything. -->

To move on, write a comment containing anything you want!
